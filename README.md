## 🛠️ Tecnologias Utilizadas

O projeto foi construído utilizando tecnologias web fundamentais para garantir performance e simplicidade:

- **HTML5:** Estrutura semântica do site.
- **CSS3:** Estilização moderna com Grid e Flexbox, incluindo design responsivo.
- **JavaScript:** Interatividade e manipulação de DOM.
- **Google Fonts:** Tipografia selecionada (Anybody e Newsreader).
- **GitHub Pages:** Hospedagem automatizada.

## Como executar localmente

Se desejar clonar e rodar o projeto em sua máquina:

1. Clone o repositório:
   ```bash
   git clone https://github.com/vrodrigoleite/portifolio.git
   ```
2. Navegue até a pasta:
   ```bash
   cd portifolio
   ```
3. Abra o arquivo `index.html` em seu navegador.
